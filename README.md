# nodejs-crawler

`nodejs` 抓取页面数据，批量下载图片表情等

## getEmoji-爬取表情

```js
npm install
node getEmoji.js
```

两种不同的方式爬取表情，批量下载图片保存到本地, 方法一 :  `node-request`, 方法二 : `node-crawler`
